Szükséges fájlok formátuma:
///////////////////////////////////////

init: 

SSID(hálózat neve)
Password
szerever ip cím 
kommunikációs port(must port>1024!)

///////////////////////////////////////

progs:

5db programot lehet definiálni(program korlátai miatt)

24 db adat tesz ki egy programot, ennek decimális számnak kell lenni. 
Szóköz enter vagy tabbal elválaztva.
Ha csak egy program lesz, a maradék 4*24=96 számot ismételd meg azzal azzal az egyel amit használni szeretnél.

///////////////////////////////////////

save:

legutobb futott program száma
hőmérséklet offset
hordozható egység hőmérséklet offsete

///////////////////////////////////////

clock:

két szám
1. 0-23 (óra)
2. 0-59 (perc)
